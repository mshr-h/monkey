# monkey
Writing An Interpreter In Go

## Installation

```
go get github.com/mshr-h/monkey
```

## Running REPL

```
cd path/to/mshr-h/monkey
go run main.go
```

Then you'll see the prompt.

```
Hello mshr-h! This is the Monkey programming language!
Feel free to type in commands
>>
```
